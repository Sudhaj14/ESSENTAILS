Features

  • Built with Godot engine.
  • Interactive gameplay with 5 levels.
  • Each level includes:
    • Terminal
    • Output screen
    • Check buttons to verify task completion
  • Attractive UI with:
    • Check buttons to see whether your tasks are completed.
    • To-do lists
    • Taskbar with:
      • Quit button
      • Help menu containing a manual to understand the concepts
      • Back arrow to toggle to the previous level
